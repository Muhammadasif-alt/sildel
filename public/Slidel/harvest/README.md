# Real Alentejo Harvest Photography

Authentic photos from Sildel's actual cork sourcing operation, supplied
by the founder. These replace the AI Nano Banana 2 placeholders where
the site narrates the harvest, origins, and sustainability story.

## Required files

Drop each photo at the exact path below — the code already references
them. WebP is preferred (smaller); JPG also works.

| # | Filename | What the photo shows | Where it's used |
|---|---|---|---|
| 1 | `harvest-team.jpg` | The crew (~10 people) posing in front of the tractor with massive stacks of harvested cork sheets behind them, blue sky | LUXlife-2026 award page — Overview |
| 2 | `harvest-axe.jpg` | Two harvesters working a cork oak — one with axe at the base, one above. Trunk freshly stripped (orange + black contrast) | LUXlife-2026 award page — Hero |

## After saving

Once all seven photos are in this folder, the site should render
correctly with no further code changes — `next dev` and the next
deploy will pick them up automatically.

If a photo is missing, Next.js renders the alt text in place of the
image (no crash), so it's safe to drop them in one at a time.
